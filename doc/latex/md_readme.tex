\subsubsection*{一、项目说明}

本项目实现了以下内容：
\begin{DoxyItemize}
\item 端到端的推理，输入图像，输出装甲板信息(坐标、四点、类别等)
\item 有效识别 部分遮挡、图案残缺 的装甲板
\item 场景泛化性强，极大减少参数调试压力
\end{DoxyItemize}



\subsubsection*{二、工作条件}

务必保证摄像头焦距正确，同时镜片干净无污物。 光照不足时，调整摄像头曝光或增益数值，直到图案有区分度。
